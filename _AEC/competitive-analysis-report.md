## 1. 현재 서비스 현황 요약

| 항목 | 현재 상태 |
|------|-----------|
| **서비스 유형** | 커스텀 일러스트 명함 제작 플랫폼 |
| **기술 스택** | Next.js 16, Supabase, Cloudflare Workers |
| **카드 요소** | 앞면(이름+일러스트), 뒷면(이름/직함/해시태그/소셜링크) |
| **관리자 기능** | CSV 일괄 업로드, 일러스트 관리, 상태 관리, 카드 편집 |
| **사용자 기능** | 카드 정보 입력, 대시보드, 수정 요청, PNG 다운로드 |
| **인증** | Supabase Auth (이메일/Google OAuth) |
| **배포** | Cloudflare Workers (Edge 배포) |

### 주요 기능 상세

**사용자 기능:**
- 개인화된 명함 디자인 (이름, 직함, 회사, 소셜 링크)
- 프로필 이미지 업로드 (드래그앤드롭, 5MB 제한)
- 앞/뒷면 카드 디자인 커스터마이징 (색상, 텍스트)
- 고품질 PNG 이미지 내보내기 (2x 해상도)
- 전용 대시보드에서 요청 상태 추적
- 카드 디자인 확인/승인

**관리자 기능:**
- 모든 카드 요청 검토 및 관리
- CSV/Excel 파일을 통한 일괄 업로드 (자동 사용자 등록)
- 카드용 커스텀 일러스트 이미지 업로드
- 요청 상태 변경 (submitted → processing → confirmed/rejected/delivered)
- 배달 전 카드 콘텐츠 편집
- 외부 이미지 URL 및 Google Drive 링크 지원

### 카드 디자인 요소

**앞면:**
- 표시 이름 (최대 40자)
- 일러스트 이미지 (전체 카드 커버)
- 텍스트 색상 (흰색+검정 스트로크 또는 검정)
- 배경색 (10가지 프리셋 한국 컬러 또는 커스텀 hex)
- 나눔명조 한국 세리프 폰트

**뒷면:**
- 전체 이름 (30px 고정)
- 직함/역할 (20px 고정)
- 해시태그 (20px, 커스터마이즈 가능)
- 소셜 링크 (platform/handle 형식)
- 소셜 링크 순서: email → phone → linkedin → instagram → facebook (우측 정렬)
- 배경색 (커스터마이즈 가능)

## 2. 시장 규모 및 트렌드

### 시장 규모

- **2025년**: USD 2.15억 달러
- **2035년 전망**: USD 6.8억 달러
- **연평균 성장률 (CAGR)**: 12.2%

### 주요 트렌드

- Fortune 500 기업 중 **93%**가 디지털 명함 솔루션 도입
- 전문직 종사자 **25~30%**가 디지털 명함 선호
- 소기업의 **37%**가 디지털 명함 통합 완료
- **NFC + Apple/Google Wallet** 통합이 차세대 표준으로 부상
- **AI 기반 기능** (명함 스캔, 리드 스코어링, 자동 후속 조치)이 프리미엄 차별화 요소

## 5. SWOT 분석

### Strengths (강점)

- **커스텀 일러스트 명함** - 시장에서 유일한 아트워크 기반 명함 서비스. 어떤 경쟁사도 개인화된 일러스트를 카드 디자인의 핵심으로 사용하지 않음
- **관리자 워크플로우** - CSV 일괄 업로드, 상태 관리, 일러스트 업로드, 원본 vs 완성본 비교 등 체계적 관리 도구
- **한국 시장 특화** - 나눔명조 폰트, 한국어 UI, 한국 컬러 팔레트, 한국 사용자 경험에 최적화
- **Edge 배포** - Cloudflare Workers로 글로벌 저지연 서비스 제공
- **모던 기술 스택** - Next.js 16, React 19, TypeScript, Tailwind CSS 4 등 최신 기술 활용

### Weaknesses (약점)

- **공유 기능 부재** - QR 코드, 링크 공유, vCard 다운로드가 없어 디지털 명함의 핵심 가치 미달
- **단일 카드 레이아웃** - 템플릿 선택 불가, 사용자 다양성 제한
- **보안 취약점** - RLS 정책 미설정(`USING (true)`), Rate Limiting 없음, 보안 헤더 미적용
- **알림 시스템 없음** - 상태 변경 시 이메일/푸시 알림 부재
- **모바일 앱 없음** - 웹 전용 서비스로 접근성 제한
- **분석 기능 없음** - 카드 조회수, 클릭 통계 등 인사이트 부재

### Opportunities (기회)

- **한국 시장 공백** - 디자인 중심 디지털 명함 서비스가 부재한 명확한 시장 기회
- **기업 B2B** - 기업 임직원 명함 일괄 제작 수요 (현재 CSV 업로드 기반으로 확장 가능)
- **AI 통합** - 사진 기반 일러스트 자동 생성으로 핵심 차별점 확장
- **이벤트/네트워킹** - 컨퍼런스, 행사, 네트워킹 이벤트에서의 명함 교환 시장
- **카카오톡 생태계** - 한국 최대 메신저를 통한 명함 공유 기능

### Threats (위협)

- **리멤버** - 한국 1위 프로페셔널 네트워크의 디지털 명함 기능 확장 가능성
- **글로벌 서비스** - Popl, Blinq 등의 한국 시장 진출 및 한국어 지원
- **AI 디자인 도구** - Canva, Midjourney, DALL-E 등으로 직접 명함 디자인 가능
- **Apple/Google 자체 솔루션** - 빅테크의 디지털 명함 기능 내장 가능성

## 6. 개선방안 (우선순위별)

### Phase 1: 필수 기능 보강 (즉시 실행)

| # | 개선 항목 | 설명 | 기대 효과 | 난이도 |
|---|----------|------|-----------|--------|
| 1 | **QR 코드 공유** | 각 명함에 고유 QR 코드 생성, 공유 가능한 웹 링크 | 시장 표준 기능 확보 | 중 |
| 2 | **vCard 다운로드** | 명함 조회 시 연락처 저장 버튼 (.vcf 파일 다운로드) | 실용성 대폭 향상 | 하 |
| 3 | **보안 강화** | RLS 정책 적용, Rate Limiting, 보안 헤더(CSP, HSTS) 추가 | 프로덕션 안정성 확보 | 중 |

### "아트워크 명함" 전략

현재 서비스의 **최대 강점**은 **커스텀 일러스트 기반 명함**입니다. 이는 전 세계 디지털 명함 시장에서 유일무이한 포지션입니다.

#### 전략 방향

1. **"세상에 하나뿐인 명함"** - 아트워크 명함으로 프리미엄 포지셔닝
   - 개인화된 일러스트는 사진이나 템플릿과 차원이 다른 가치 제공
   - 명함을 단순 연락처 교환 도구가 아닌 "개인 브랜딩 아트워크"로 재정의

2. **AI 일러스트 자동 생성** 추가 시 확장성 극대화
   - 현재 수동 일러스트 제작의 병목 해소
   - 대량 주문 처리 가능
   - 다양한 아트 스타일 제공 (수채화, 펜화, 팝아트 등)

3. **QR/공유링크 + 분석** 추가로 시장 표준 기능 확보
   - 일러스트 명함의 아름다움을 디지털로 공유할 수 있는 채널 확보
   - 분석 기능으로 명함의 비즈니스 가치 증명

4. **카카오톡 + Apple Wallet** 통합으로 한국 시장 킬러 조합 완성
   - 카카오톡: 한국 사용자의 일상적 공유 채널
   - Apple Wallet: 폰 더블탭으로 즉시 명함 공유 (Apple Pay처럼)
